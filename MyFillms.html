Myfilms es la típica tienda de películas con un diseño útil i sencillo, las películas se pueden puntuar y comentar. puedes realizar el pago de las películas a través de PayPal. También cuenta con un panel administrativo con un CRUD para películas, usuarios y categorías.

Front-end

Este sitio web tiene un diseño simple y atractivo, está diseñado con HTML, CSS y Bootstrap. Utiliza AJAX para algunas comunicaciones con el servidor y devuelve una respuesta con una notificación (toastr). También usa modales que se hacen con la ayuda de bootstrap en la parte de administracion.

Tecnologias usadas

Bootstrap, Jquery, AJAX, JSON, font-awesome, toastrJS.

Back-end

La tecnología principal de toda esta aplicación es Laravel que es un framework de PHP. Usé MySql, una base de datos que funciona muy bien con laravel. También se usan sesiones que sirven para hacer el carro y que lo que hay no se puede perder.

Tecnologias usadas

PHP, MySql, Laravel, PayPal API.
