Myfilms it's the typical films store with a simple and useful design, The films can be rated and commented. you can make the payment of the movies through PayPal. It also has an administrative panel with a CRUD for movies, users and categories.

Front-end

This website has a simple and attractive design, it is designed with HTML, CSS and some Bootstrap. It uses AJAX for some communication with the server and returns a response with a notification (toaster). Also use modals that are done with the help of bootstrap in the admin part

Used Technologys

Bootstrap, Jquery, AJAX, JSON, font-awesome, toastrJS

Back-end

The main tecnology of this whole application in Laravel that it's a PHP framework. I've used MySql a database that works very good with laravel. Sessions are also used to make the car and that what there is can not be lost

Used Tecnologys

PHP, MySql, Laravel, PayPal API
