Este proyecto es una aplicación web de fitness destinada a asesores que hacen asesoramiento online. Facilita la comunicación entre clientes y asesores, mejorar la organización y la eficiencia del asesor. Busca motivar a los clientes con estadísticas y datos de su progreso. Solo está disponible en catalán.

Front-end

Diseñé este sitio web con una interfaz de usuario fácil e intuitiva, es totalmente responsive. He usado HTML y CSS para diseñar algunas partes de la web, con la ayuda de bootstrap para diseñar y hacerlo completamente responsive. He usado AJAX en algunos formularios para pasar datos y recibir una respuesta con una notificación (toastr), se hizo pasando datos entre el front-end y back-end con objetos JSON, todo eso fue con la ayuda de Jquery.

Tecnologias usadas

Bootstrap, Jquery, AJAX, JSON, font-awesome, toastrJS

Back-end

La principal tecnología de toda esta aplicación es Laravel, que es un framework de PHP. Usé MySql, una base de datos que funciona muy bien con laravel. La API de paypal se usa para hacer los pagos de la aplicación entre los clientes y sus asesores.

Tecnologias usadas

PHP, MySql, Laravel, PayPal API
