React Projects

Este proyecto es una aplicación web de fitness destinada a asesores que hacen asesoramiento online. Facilita la comunicación entre clientes y asesores, mejorar la organización y la eficiencia del asesor. Busca motivar a los clientes con estadísticas y datos de su progreso. Solo está disponible en catalán.

Front-end

Aplicación utilizando React y Redux con un disenyo simple, cada proyecto tiene un disenyo aparte.

Tecnologias usadas

React, Redux, CSS.

Back-end

Utilize una API de Laravel para el Backend del proyecto la API utiliza una base de datos MySql. El inicio de session funciona utiliza el paquete de Passport

Tecnologias usadas

PHP, MySql, Laravel, Passport
